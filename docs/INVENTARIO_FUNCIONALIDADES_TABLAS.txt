================================================================================
RESUMEN DE HERRAMIENTAS DE PERSONALIZACIÓN GLOBALES
================================================================================

COLUMNAS & VISIBILIDAD
- useColumnasOcultas .......... almacena columnas ocultas en localStorage
                                (key = "modulo-cols-vN")
- ColumnasToggleButton ........ UI desplegable para mostrar/ocultar columnas;
                                soporta columnas "obligatorias" no ocultables
- visibleColumns(cols, ocultas) filtro que aplica las ocultas
- useColumnasRedimensionables . drag en el header para ajustar ancho; persiste
                                widths en localStorage (-widths-vN)

FILTROS
- filtroPorColumna(rows, key) . helper que arma filters[] con valores únicos
- RangoFechasFiltro ........... DatePicker desde/hasta para una fecha
- useRangoFechasPersistente ... rango con persistencia en localStorage
- usePersistedState ........... custom hook para guardar cualquier estado en LS
- Segmented ................... control tipo tab para quick filters
- Input search, Select, DatePicker en cards superiores
- dentroDeRango(fecha, rango) . validador

PAGINACIÓN
- paginacionEstandar({ ... }) . config con page-size selector (default 50)
- PAGINATION_PAGE_SIZE ........ constante = 50

ORDENAMIENTO
- sorter: true (server-side) o sorter: (a,b) => ... (client-side)

EXPORT
- ExportarExcelButton ......... genera XLS:
                                · respeta filtros server-side
                                · respeta layout (orden + ocultas)
                                · category filters dropdown
                                · date filter
                                · currentRows para exportar solo lo visible
- DescargarOTExcelButton ...... export específico de una OT (externa/interna)

EDICIÓN
- EditableCell ................ cell inline editable (string/number/select)
- Modal Form .................. edición multi-campo
- Draft pattern ............... cambios locales + Save por fila o bulk

SELECCIÓN MASIVA
- rowSelection (Ant Design) ... checkboxes + bulk actions

OTROS
- STICKY_HEADER ............... headers fijos al scrollear
- useTabSync .................. sync cross-tab (refresh en otra pestaña)
- useEditLock ................. lock pesimista server-side
- formatDateOnly, numeracionColumn, brand tokens (theme)

TOTALES Y NOTAS FINALES

HERRAMIENTAS DE PERSONALIZACIÓN: 12+
  · Columnas ocultas + toggle + persistencia
  · Drag-to-resize columnas + persistencia
  · Drag-to-reorder filas (templates de reqs)
  · Filtros (server-side + client-side, multi-tipo)
  · Rangos de fecha persistentes
  · Búsqueda libre
  · Segmented quick-filters
  · Paginación con page-size selector
  · Sticky headers
  · Sortable columns (server + client)
  · Edición inline (cells)
  · Selección masiva + bulk actions
  · Export Excel respetando layout + filtros
  · Import Excel (clientes, proveedores, materiales)
  · Modal forms para CRUD
  · Locks pesimistas server-side
  · Tab sync cross-tab
  · localStorage persistence por usuario

NOTAS:
- Todos los listados usan AntD Table con scroll horizontal x="max-content"
- La mayoría tienen toolbar superior con búsqueda + filtros + botones
- Los filtros enum se generan automáticamente con `filtroPorColumna(rows, key)`
- Los formatos de fecha son consistentes (DD/MM/YY o DD/MM/YYYY)
- Tags con colores brand (navy, cyan, purple, etc.) — todo desde theme.ts
- Responsive: las tablas usan scroll horizontal; modales usan modalWidth()
- Edit locks evitan colisiones en planificación y OTs externas
- Auditoría completa: cambios en OT/Req/OC quedan en OTHistorial
- localStorage keys versionados (vN) para invalidar caché al cambiar columnas
